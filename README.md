# travian-bot
travian bot made with python

## first
make sure that the game server allow to us bot in the game (try private servers)

## Requirements
* Python 3.8
* bs4 module
* selenium module
* re module
* subprocess module
* chrome and chromedriver

## Installation
* install python
* pip install selenium bs4 

## How to run?
* go into the travian_bot.py file with an editor
* build the control game tasks and save
* run start_bot.file
